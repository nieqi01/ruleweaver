## 使用说明

### 1. 创建副本
在"实体库"页面点击"新建副本"，输入副本名称、危险等级等信息。

### 2. 添加实体
选择副本后，可以添加角色、道具、规则、地图等实体。

### 3. 建立关系
在"关系图谱"页面，选择两个实体并建立关系（冲突、伏笔、因果、持有）。

### 4. 地图标记
在"地图"页面上传平面图，并在上面标记实体位置。

### 5. 章节体检
在"章节体检"页面粘贴章节内容，系统会自动检测：
- 未定义的实体引用
- 状态矛盾（如死亡角色出场）
- 伏笔的"埋-收"闭环

### 6. AI 辅助
在"AI中心"页面：
- 设置 Kimi API Key
- 评估规则合理性
- 生成角色/规则

### 7. 数据同步
在"设置"页面导出/导入世界设定包，支持加密和差异合并。

## 文本标记语法

在章节体检中支持以下标记：

```
{{角色:张三}}          - 引用角色
{{道具:染血的校规}}     - 引用道具
[[染血的校规]]         - 道具简写
<<照片里的手:埋>>      - 埋下伏笔
<<照片里的手:收>>      - 回收伏笔
```

## 离线使用

1. 首次访问应用后，浏览器会自动缓存核心资源
2. 之后即使无网络也能正常使用
3. 数据存储在本地 IndexedDB 中
4. AI 功能需要联网

## 数据安全

- 所有数据存储在本地浏览器中
- 导出时可选择 AES-256 加密
- API Key 仅存储在 localStorage，不上传服务器
